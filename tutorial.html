📌 Important Notes

  • ✅ Linear Search: Does not require sorted data. Suitable for small or simple cases.
  • ✅ Binary Search: Very fast but requires the array to be sorted beforehand.
  • ✅ Merge Sort vs Quick Sort:
    • Merge Sort always runs in O(n log n), but needs extra space.
    • Quick Sort is usually faster, but worst-case performance is O(n²).

Sample Python Functions

Here are some sample Python functions demonstrating basic algorithms:

1. Bubble Sort

This function sorts an array using the Bubble Sort algorithm.


def bubbleSort(arr):
    n = len(arr)

    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]

2. Linear Search

This function searches for a target element in an array using Linear Search.


def linearSearch(arr, target):
    n = len(arr)
    for i in range(n):
        if arr[i] == target:
            return i
    return -1

3. Find Largest Number

This function finds the largest number in an array.


def largestNumber(arr):
    n = len(arr)
    largestNumber = arr[0]

    for i in range(1, n):
        if arr[i] > largestNumber:
            largestNumber = arr[i]
    return largestNumber

4. Find Smallest Number

This function finds the smallest number in an array.


def lowestNumber(arr):
    n = len(arr)
    lowestNumber = arr[0]

    for i in range(1, n):
        if arr[i] < lowestNumber:
            lowestNumber = arr[i]
    return lowestNumber

Example Usage

Here's how you can use these functions:


arr = [60, 50, 20, 10, 40]
print(largestNumber(arr))  # Output: 60
